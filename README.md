# 重力隧道 · Gravity Tunnel Runner

## 简介
重力隧道是一款3D跑酷游戏，玩家需要在隧道中躲避障碍物，通过切换墙壁和跳跃来前进。游戏支持PC和移动端操作，并提供了多种难度选择。

## 特性
- **游戏玩法**：
  - **PC端**：使用方向键（←/→/↓）切换墙壁，空格或↑键跳跃。
  - **移动端**：通过滑动屏幕来切换墙壁，向上滑动跳跃。
- **用户界面**：
  - 游戏开始、暂停和结束时的界面。
  - 显示分数、速度、能量条等信息的HUD。
  - 成就系统和排行榜。
  - 音频控制（背景音乐和音效开关）。
  - 难度选择（简单、普通、困难）。
- **视觉效果**：
  - 使用Three.js库进行3D渲染。
  - 包含多种视觉特效，如粒子效果、光晕等。
- **音频系统**：
  - 背景音乐和音效管理。
  - 支持音量调节。

## 技术栈
- HTML5
- CSS3
- JavaScript
- Three.js (用于3D渲染)

## 如何运行
1. 打开 `code.html` 文件即可开始游戏。
2. 通过点击界面上的不同选项来选择难度、查看成就或排行榜。
3. 使用键盘或触摸屏进行游戏操作。
